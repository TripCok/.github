# **Tripcok**

## **📌 Project Goal**
현대 여행자들을 위해 개인 성향과 실시간 데이터를 기반으로 한 맞춤형 여행지 추천 서비스를 제공하며, 그룹 구성원의 공통 관심사를 반영해 모두가 만족할 수 있는 그룹 기반 추천 기능도 함께 제공합니다.  

이 서비스는 개인의 여행 경험을 향상시키는 동시에 여행 커뮤니티를 활성화하여 사용자 간의 소셜 연결과 공동 여행 계획을 보다 효율적으로 지원합니다.  
또한, 관리자 페이지를 통해 추천 모델의 성능과 데이터 상태를 실시간으로 모니터링할 수 있어 정확한 의사결정을 돕는 통합 관리 환경을 제공합니다.
